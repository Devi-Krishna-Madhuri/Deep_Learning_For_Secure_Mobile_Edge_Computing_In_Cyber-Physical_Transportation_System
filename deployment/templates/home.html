Deep Learning for Secure Mobile Edge Computing in Cyber-Physical Transportation Systems

This system enhances security in mobile edge computing devices. With the rise in cyber threats, we developed a system that detects different types of attacks on mobile edge computing devices based on network parameters. If an attack is detected, an alert message is sent to the targeted user via email.
